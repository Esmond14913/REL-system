HQA Portal - Release History / 版本變更歷史紀錄
==================================================

v1.5.2 (Latest) - 2026-04-22
----------------------------
[EN]
- Redesigned Unit Converter with a high-density Matrix Layout.
- Implemented multi-way synchronized unit conversion (Length, Force, Mass).
- Added bi-directional input support with yellow highlighted focus areas.
- Integrated "Conversion Ratio" labels for engineering transparency.
- Removed legacy category switchers for a more streamlined experience.
[ZH]
- 徹底重構單位換算器，採用高密度「矩陣式佈局」。
- 實現長度、力量、重量三組單位的「多向同步換算」邏輯。
- 新增黃色手動輸入區，支援組內任一單位輸入後全自動同步。
- 整合「換算比例」標籤，提升工程計算透明度。
- 移除舊版分類切換按鈕，優化操作流程。

v1.5.1 - 2026-04-22
----------------------------
[EN]
- Consolidated "Arrhenius AF" and "Success Test Planner" into a single "MTBF Calculator".
- Redesigned MTBF interface into a side-by-side high-density dashboard.
- Upgraded statistical logic to support Failure Counts (c >= 0) using Chi-Square Inverse.
- Added "Assessment Mode" with individual unit test duration tracking.
- Implemented "Calculation Summary & Purpose" dynamic card (Bilingual).
- Optimized all UI labels with bilingual (EN/ZH) descriptions and 15px fonts.
[ZH]
- 整合「加速因子」與「成功試驗計畫」為單一「MTBF 計算器」。
- 採用並排式 (Side-by-Side) 高密度儀表板設計。
- 升級統計邏輯，透過卡方分布反函數支援「失效數 (c >= 0)」計算。
- 新增「結果評估模式」，支援個別樣本測試時數輸入與自動加總。
- 新增「計算摘要與目的」動態生成卡片（中英雙語）。
- 全介面標籤雙語化，並將輸入標題優化為 15px 字級。
